Panduan Penggunaan

1. Selamat Datang!

Aplikasi ini dirancang khusus untuk mengekstrak, mendekripsi, dan menerjemahkan file teks dari game retro J2ME (Gameloft) secara mudah.

/

2. Metode Online & Offline

Gunakan API Key Gemini (gratis) di Pengaturan untuk terjemahan otomatis AI.

Tanpa API Key (Offline)? Anda tetap bisa menggunakan alat ini sepenuhnya untuk mengedit teks secara manual dan memanfaatkan fitur Kamus Lokal.

3. Pilih & Muat File

Pilih file game yang ingin diproses. Jika file Anda berformat Decryptor 01/Decryptor 02 ganda, buka Pengaturan dan aktifkan Impor Teks Asli (EN) agar dapat membandingkan struktur file terjemahan.

4. Cara Mengedit Teks

Ketuk teks mana saja untuk membuka jendela Edit. Anda bisa mengetik manual terjemahan Anda, atau klik tombol Terjemahkan (AI). Aktifkan ikon Gembok jika Anda butuh memastikan panjang karakter teks sama persis dengan aslinya.

5. Mode Pilih Banyak

Tekan & Tahan (Long Press) pada sebuah teks untuk masuk ke mode multi-select. Pilih beberapa teks sekaligus, lalu gunakan tombol melayang di pojok kanan bawah untuk menerjemahkan secara massal.

6. Fitur Kamus (Dictionary)

Untuk terjemahan konsisten tanpa AI, simpan istilah yang sering muncul ke Kamus. Aplikasi akan otomatis mengganti kata-kata tersebut di seluruh paragraf saat Anda melakukan Replace All atau saat AI memprosesnya.

7. Simpan & Unduh

Jangan lupa menyimpan progres Anda di menu Berkas (Files) agar tidak hilang saat ditutup. Jika sudah selesai semua, klik ikon Download melayang berwarna hijau untuk mengunduh file hasil (ZIP/Biner) yang siap di-inject ke game.
